## Project Setup

```sh
$ pnpm install

# Compile and Hot-Reload for Development
$ pnpm dev


# Compile and Minify for Production
$ pnpm build


# Lint with ESLint
$ pnpm lint

# Run vitest tests
$ pnpm test:unit
```

For a detailed explanation on how things work, checkout [Vue.js docs](https://vuejs.org/).
