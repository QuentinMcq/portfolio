## Project Setup

```sh
$ yarn install

# Compile and Hot-Reload for Development
$ yarn dev


# Compile and Minify for Production
$ yarn build


# Lint with ESLint
$ yarn lint

# Run vitest tests
$ yarn test:unit
```

For a detailed explanation on how things work, checkout [Vue.js docs](https://vuejs.org/).
